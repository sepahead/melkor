## reader-budgets (P0-12 slice) — completed; decoded-SPZ residual stays with P0-09

PLY now charges input bytes, header bytes, declared splats, and canonical reconstructed storage
before allocation. SPZ charges compressed input and canonical decoded storage; its vendored v1–v3
codec still performs a whole-stream inflate internally, so the tighter pre-inflate decoded-size cap
remains explicitly coupled to the P0-09 SPZ v4/zstd re-pin. Tests `ply_budget_tests` and
`spz_budget_tests` pass in all A1 matrices. The original scope below is retained as history.
